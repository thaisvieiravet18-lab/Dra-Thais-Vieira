# Meu Primeiro Pet - Nutrição Veterinária 🐾

Site profissional e otimizado da **Dra. Thais Vieira**, Médica Veterinária especialista em Nutrologia e Longevidade de pets. O projeto conta com páginas interativas, detalhes de serviços (Teleconsulta, Consultório, Convênios), área exclusiva do plano "Meu Primeiro Pet" e integração fluida de canais de atendimento (WhatsApp e Instagram).

---

## 🚀 Tecnologias Utilizadas

- **React 18** (com TypeScript)
- **Vite** (Build tool ultra-rápido)
- **Tailwind CSS** (Estilização moderna e responsiva)
- **Framer Motion** (Animações de alto padrão)
- **Lucide React** (Ícones modernos)
- **Express** (Para backend/full-stack opcional)

## 🛠️ Como Executar o Projeto Localmente

Se você baixou o projeto em formato **ZIP**, siga estes passos para rodá-lo no seu computador:

### Pré-requisitos
Você precisará ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

### Passos:
1. **Extraia o arquivo ZIP** em uma pasta de sua escolha.
2. **Abra o terminal** (Prompt de Comando ou VS Code Terminal) nessa pasta.
3. **Instale as dependências** rodando:
   ```bash
   npm install
   ```
4. **Inicie o servidor de desenvolvimento**:
   ```bash
   npm run dev
   ```
5. Abra o navegador no endereço indicado (geralmente `http://localhost:3000` ou `http://localhost:5173`).

---

## 📦 Como Compilar para Produção

Para gerar a versão otimizada final do site:
```bash
npm run build
```
Os arquivos prontos para publicação serão gerados dentro da pasta `dist/`.
